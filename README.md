=====================

Parser for graph files in [DOT format](https://en.wikipedia.org/wiki/DOT_(graph_description_language)).

Installation
------------

```
git clone https://github.com/phovea/phovea_data_graph_dot.git
cd phovea_data_graph_dot
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
